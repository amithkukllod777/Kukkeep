## Headline takeaways

1. **Biggest gap:** no offline support — every one of the three competitors
   verified as offline-capable (or reasonably assumed so for Keep); KukKeep
   requires a live connection for every action.
2. **Second biggest gap:** Android-only — every competitor covers iOS.
3. **Clearest strength:** a native, free, in-app AI-over-your-notes feature —
   none of the three competitors ship the exact same free/native combination
   (Keep's AI is Gemini-app-mediated and not free-vs-paid-differentiated in
   the same way; Evernote's is native but paid-gated; OneNote's is
   Copilot-gated).
4. **Second clearest strength:** genuinely free with no caps, unlike
   Evernote's restrictive 50-note/1GB free tier.
5. **Easiest wins:** fix the view-scoped search (BUG-003) and add basic
   import/export — both are Table-Stakes gaps with moderate-or-lower
   implementation cost relative to offline support or iOS.
